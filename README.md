# cheonan_data-
천안시 공공데이터 경진대회

주제1: 천안시 학교 그린통학로 구축 필요성 분석 
- 충청남도에서 천안시가 교통사고 사상자 현황이 가장 높았으며, 그 중 '안전의무불이행'에 따른 사고가 가장 높았음. 
- 천안시는 작년대비 어린이 교통사고현황이 증가하였기에, 교통사고 감소를 위한 그린통학로 구축의 필요성을 말하고자 함. 

[분석항목(데이터)]
- 천안시 초,중,고 위치 데이터 
- 천안시 도로교통량 데이터 
- 천안시 교통사고량 데이터 

[분석아이디어] 
- 학생 통학경로의 교통량 노출 수준 
- 어린이 보호구역 내 사고 빈도수, 현황

[정책제안]
- 통학로 차량 통제 시간대 지정
- 학교 주변 자전거 전용도로 확충
- 친환경 셔틀버스 시범도입
